# Graybeard Installed

Graybeard is active in `balanced` mode by default.

Commands:

- `/graybeard advisory`
- `/graybeard balanced`
- `/graybeard strict`
- `/graybeard off`
- `/graybeard-review`
- `/graybeard-audit`
- `/graybeard-help`

Bundled skills are available as `graybeard:graybeard`,
`graybeard:graybeard-review`, `graybeard:graybeard-audit`, and
`graybeard:graybeard-help`.
